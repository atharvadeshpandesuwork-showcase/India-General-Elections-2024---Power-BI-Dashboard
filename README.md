## India General Election Results 2024 - Power BI Dashboard

### Project Overview

The India General Election Results 2024 dashbord is designed to provide a comphrehensive analysis of election outcomes accross India. The objective is to visualize alliance performance, state-wise seat distribution, constituency level results, and political trends through interactive Power BI dashboards. This project enables political analysts, researchers, journalists, and citizens to explore election results at national, state, and constituency levels using dynamic visualizations and drill trhough analysis.

---

### Business Objective

The primary goal of this project is to:
- Analyze seat distribution among major political alliances.
- Compare performance of NDA, I.N.D.I.A, and other parties.
- Indentify state-level political dominance.
- Explore contituency-wise election outcomes.
- Provide an interactive platform for election result exploration.
- Enable data-driven insights through visual storytelling and drill-through analysis.

---

## Dashboard Modules

### Overview Analysis
Provides high level summary of elelction results accross India.

#### Key Metrics
- Total seats won by NDA Alliance.
- Total seats won by I.N.D.I.A Alliance.
- Total seats won by Independent and Other Parties.
- Percentage share of seats won by each alliance.

#### Analysis
- Party-wise seat count within NDA.
- Party-wise seat count within I.N.D.I.A.
- Alliance-wise comparison of election performance.
- Detailed drill-through tables for all winning constituencies.

---

### State Demographic Analysis
Provides a geographical view of election outcomes across states.

#### Key Insights
- Total seats by state.
- Majority alliance in each state.
- NDA seat count by state.
- I.N.D.I.A count by state.

#### Visualizations
- State-wise map analysis.
- Constituency-level bubble maps.
- Majority alliance identification by state.
- Drill-through state-level details.

---

### Political Landscape by state
Analyzes political performance within a selected state.

#### Key Metrics
- Seats won by NDA.
- Seats won by I.N.D.I.A
- Seats won by Others.

#### Analysis
- State-wise party performance.
- Party seat share distribution.
- Alliance dominance comparison.
- Interactive state map exploration.

---

### Constituency Analysis
Provides setailed insights for individual contituencies.

#### Key Metrics
- Total votes cast.
- EVM Votes.
- Postal Votes.
- Number of candidates.

#### Candidate Analysis
- Winning Candidate
- Runner Up Candidate
- Second Runner Up - Candidate 
- Vote Share Percentage
- Winning Margin

---

### Election Details Grid

#### Information Included
- Contituency Name
- Winning Candidate
- Runner-Up-Candidate
- Winning Party
- Alliance classification
- EVM Votes
- Postal Votes
- Total Votes

#### Features
- Drill-through functionality.
- Data Export to excel.
- Bookmark driven "Show all data" option.
